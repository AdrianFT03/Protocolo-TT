\section{Requerimientos del sistema}
Las definiciones de los requerimientos especifican las funciones y propiedades esenciales y deseables que el sistema debe cumplir.\\

A continuación se muestran los requerimientos que deberá cubrir el sistema al finalizar el proyecto para garantizar el cumplimiento de los objetivos planteados.
\begin{ReqSist}
	
	\reqSistItem{RS-R1}{El sistema enviará notificaciones con la información reelevante del percanse}{Alta}{Funcional}

	\reqSistItem{RS-R2}{El sistema implementará una aplicación movil para la gestión de usuarios}{Alta}{Funcional}

	\reqSistItem{RS-R3}{El sistema implementará una aplicación movil para la gestión de notificaciones}{Alta}{Funcional}
	
	\reqSistItem{RS-R4}{El sistema implementará una aplicación movil para la gestión de contactos}{Alta}{Funcional}
	
	\reqSistItem{RS-R5}{El sistema implementará una aplicación movil para la gestión de vehiculos}{Alta}{Funcional}
	
	\reqSistItem{RS-R6}{El sistema implementará una aplicación movil para la gestión de datos personales}{Alta}{Funcional}
	
	\reqSistItem{RS-R7}{El sistema almacenará la información de la cuenta del usuario}{Alta}{Funcional}
	
	\reqSistItem{RS-R8}{El sistema utilizará información del dispositivo para el envío de notificaciones}{Alta}{Funcional}
	
	\reqSistItem{RS-R9}{El sistema utilizará información del vehículo para el envío de notificaciones}{Alta}{Funcional}
		
\end{ReqSist}
